options(info, rformat='table', pretty_table=True, pretty_print=True, overlap_only=False)[source]¶

An easy inferface to explore all of the terminology EasyMoney understands as well the dates for which data is available.

Parameters:
  • info (str) – ‘exchange’, ‘inflation’ or ‘all’ (‘all’ requires rformat is set to ‘table’)
  • rformat (str) – ‘table’ for a table or ‘list’ for just the currecy codes, alone. Defaults to ‘table’.
  • pretty_table (bool) – prettifies the options table. Defaults to True.
  • pretty_print (bool) – if True, prints the list or table. If False, returns the list or table (as a Pandas DataFrame). Defaults to True.
  • overlap_only (bool) – when info is set to ‘all’, only keep those rows for which exchange rate and inflation data overlap.
Returns:

DataFrame of Currency Information EasyMoney Understands.
Return type:

Pandas DataFrame
Raises:

ValueError – if a. rformat is not either ‘list’ or ‘table’ or b. overlap_only is True while info is not ‘all’ and rformat is not ‘table’.
